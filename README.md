## 服务器测速脚本<a name="服务器测速脚本"></a>

官网：bench.sh（秋水逸冰大佬）
使用以下命令来查看Linux系统信息，测试网络带宽及硬盘读写速率
```bash
   wget -qO- bench.sh | bash
```
或者
```bash
   curl -Lso- bench.sh | bash
```
国内三网测速脚本
```bash
   bash <(curl -Lso- https://raw.githubusercontent.com/BlueSkyXN/SpeedTestCN/main/superspeed.sh)
```
